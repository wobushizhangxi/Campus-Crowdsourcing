# Android Test APK Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a side-loadable Android test APK for `离着远` that bundles the existing React frontend, connects to `https://gather-break-long-que.trycloudflare.com`, and removes old Windows release artifacts from `release`.

**Architecture:** Keep the existing `Vite + React` frontend in `campus-frontend`, add an Android-specific build mode plus a Capacitor shell, and drive packaging from root PowerShell scripts. The Android build flow will clean Windows installer outputs, build the frontend with a fixed public API base URL, sync the web assets into a Capacitor Android project, compile a debug APK, and copy the final artifact into `release`.

**Tech Stack:** React 19, Vite 8, Axios, Vitest, Capacitor 7, Android Gradle plugin generated by Capacitor, PowerShell

```js
// campus-frontend/src/config/apiBaseUrl.js
export const normalizeApiBaseUrl = (value = '') => value.trim().replace(/\/$/, '');

export const requireConfiguredApiBaseUrl = (value) => {
  const normalizedValue = normalizeApiBaseUrl(value ?? '');

  if (!normalizedValue) {
    throw new Error('VITE_API_BASE_URL is required for Android builds.');
  }

  return normalizedValue;
};

export const resolveApiBaseUrlCandidates = ({
  configuredApiBaseUrl = '',
  isDev = false,
} = {}) => {
  const normalizedConfiguredValue = normalizeApiBaseUrl(configuredApiBaseUrl);

  if (normalizedConfiguredValue) {
    return [normalizedConfiguredValue];
  }

  if (isDev) {
    return ['http://127.0.0.1:8080', 'http://127.0.0.1:8081'];
  }

  return [''];
};
```

- [ ] **Step 2: Create a release cleanup script limited to the `release` directory**

```powershell
# cleanup-release-artifacts.ps1
[CmdletBinding()]
param(
    [string]$ReleaseDir = (Join-Path $PSScriptRoot 'release')
)

$ErrorActionPreference = 'Stop'

$resolvedReleaseDir = [System.IO.Path]::GetFullPath($ReleaseDir)
$allowedRoot = [System.IO.Path]::GetFullPath((Join-Path $PSScriptRoot 'release'))

if ($resolvedReleaseDir -ne $allowedRoot) {
    throw "Cleanup is restricted to $allowedRoot"
}

$patterns = @('*.exe', '*portable.zip', '安装说明-*.md')
$paths = @(
    (Join-Path $resolvedReleaseDir 'app-image'),
    (Join-Path $resolvedReleaseDir 'jpackage-input')
)

foreach ($pattern in $patterns) {
    Get-ChildItem -Path $resolvedReleaseDir -Filter $pattern -Force -ErrorAction SilentlyContinue |
        Remove-Item -Force -Recurse
}

foreach ($path in $paths) {
    if (Test-Path $path) {
        Remove-Item -LiteralPath $path -Force -Recurse
    }
}
```

- [ ] **Step 3: Create the Android build script that cleans, builds, syncs, compiles, and publishes the APK**

```powershell
# build-android.ps1
[CmdletBinding()]
param(
    [string]$ApiBaseUrl = 'https://gather-break-long-que.trycloudflare.com'
)

$ErrorActionPreference = 'Stop'

$rootDir = Split-Path -Parent $MyInvocation.MyCommand.Path
$frontendDir = Join-Path $rootDir 'campus-frontend'
$releaseDir = Join-Path $rootDir 'release'
$apkSource = Join-Path $frontendDir 'android\app\build\outputs\apk\debug\app-debug.apk'
$apkTarget = Join-Path $releaseDir '离着远-android-test.apk'

& (Join-Path $rootDir 'cleanup-release-artifacts.ps1')

New-Item -ItemType Directory -Path $releaseDir -Force | Out-Null

Push-Location $frontendDir
try {
    if (-not (Test-Path (Join-Path $frontendDir 'node_modules'))) {
        & npm.cmd install
    }

    $env:VITE_API_BASE_URL = $ApiBaseUrl

    & npm.cmd run build:android-test
    & npm.cmd run android:sync

    Push-Location (Join-Path $frontendDir 'android')
    try {
        & .\gradlew.bat assembleDebug
    } finally {
        Pop-Location
    }
} finally {
    Remove-Item Env:VITE_API_BASE_URL -ErrorAction SilentlyContinue
    Pop-Location
}

if (-not (Test-Path $apkSource)) {
    throw "APK was not generated: $apkSource"
}

Copy-Item -LiteralPath $apkSource -Destination $apkTarget -Force
Write-Host "APK: $apkTarget"
```
