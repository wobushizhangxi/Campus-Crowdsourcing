# Release Branding

This directory contains the default branding assets for packaged desktop releases.

- Product name: `离着远`
- Vendor: `李哲远团队`
- Primary color direction: cyan-blue
- Style: light, modern, abstract

Files:

- `app-icon.svg`: editable source artwork for the desktop icon concept
- `app-icon.png`: generated preview bitmap
- `app-icon.ico`: Windows installer and shortcut icon
